## Panel

**Extends PureComponent**

Panel

**Examples**

_Simple Panel Demo_

```javascript
class PanelDemo extends Component{
render(){
	return (
		<Panel title="Panel Title" renderRight={()=>{
			return (
				<button type="button" onClick={()=>{
					alert('pressed');
				}}>press me</button>
			);
		}}>
			<p>panel content</p>
		</Panel>
	);
}
}
```
